# Standardized Commands

This is a list of standardized commands which should be implemented as closely as possible on any given device.

## General

* TurnOff
* TurnOn

* Beep
* Blink

* Read

* Connect
* Link

## Brightness and Color

* SetBrightness
* SetBright (255/255)
* SetSoft (190/255)
* SetDim (136/255)
* SetNite (68/255)

* SetColor

## Temperature

* Auto
* Cool
* Down
* FanOff
* FanOn
* Heat
* Up
